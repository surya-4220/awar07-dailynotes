

Adv of COntainers

--> Consistent Environment : Eliminated "it works on my machine"
--> Fast Setup : clone  repo, docker run / docker compose up
--> Dependency isolation : App 1 needs python 3.9 and app 2 needs python 3.14.. - No Conflict

--> Lower Infra Wastage
--> Portability : You can move your container from anywhere to anywhere.

Docker alternative : podman / containerd / cri-o / kaniko


Dont install docker using root user.


docker --version

sudo dnf install docker -y

sudo systemctl enable docker --now

sudo systemctl status docker

sudo usermod -aG docker $USER

---

docker images					--> Lists local images
docker ps						--> Lists running contianers
docker ps -a					--> List the all container running/exited
docker run nginx:1.28			--> Runs a container with nginx image, tag 1.28 (foregound)
docker run -d nginx:1.28		--> runs nginx 1.28 in detached mode

docker run -d --name my-web nginx:1.28		--> run nginx 1.28 in detached mode with specific name

docker run --rm --name test-delete nginx	--> it removes when container stops

docker stop <container-id>/<container-name>
docker start <container-id>/<container-name>

docker kill <container-id>/<container-name>

stop : graceful (SIGTERM)
kill : immediate (SIGKILL)


docker rm <container-id>/<container-name>		--> container will be removed
